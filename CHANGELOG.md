## 0.0.1

* Initial release of Mim Chucker
* HTTP request inspector with isolate support
* Material 3 UI design
* Support for Dio, Http, and Chopper HTTP clients
* Advanced search capabilities
* In-app notifications
* JSON tree and raw view
* cURL export functionality
* Cross-platform support (Android, iOS, Web, Windows, macOS, Linux)
* Desktop platform isolate support with sqflite_common_ffi
* Error handling improvements
* Updated deprecated APIs (withValues, SharePlus)
* State management with Riverpod 3 (replaced Provider)
* Updated to Riverpod 3.0.3 with Notifier API
